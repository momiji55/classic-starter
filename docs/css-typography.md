
# CSS フォントサイズ ルール

classic-starter における、文字サイズ（`font-size`）の扱い方を定めたルールです。
このテーマを土台に新規案件を始める際は、本ルールに沿って文字サイズを組みます。

## 基本方針

| 項目 | 決定 |
|---|---|
| 単位 | **`rem`**（px 固定は使わない） |
| 定義場所 | `common.css` 先頭の `:root` に `--text-*` トークンで定義 |
| 参照方法 | 各セレクタでは **`var(--text-*)` で参照**する |
| 例外 | 親基準で相対変化させたい箇所のみ `em`（例: コードの `0.9em`）。`inherit` も可 |

### なぜこの方針か

- **rem に統一** … px 固定はユーザーのブラウザ文字サイズ設定を無視してしまう。余白（`--space-*`）が rem 基準なので、文字サイズも揃えると拡大縮小時に破綻しない。アクセシビリティで有利。
- **タイプスケール化** … 使ってよいサイズを限定することで、文字サイズの乱立を防ぎ一貫性を保つ。案件ごとのトーン調整も `:root` だけで済む。
- **`em` の例外** … インラインコードなど「親の文字サイズに対して少し小さく」したい箇所は、絶対値より相対指定（`em`）が適切なため対象外とする。

## タイプスケール

`:root` に以下を定義し、文字サイズは必ずこの中から選ぶ。

```css
:root {
  /* フォントサイズ（タイプスケール / rem 基準） */
  --text-xs: 0.875rem;  /* 14px: キャプション・フッター */
  --text-sm: 0.95rem;   /* ~15px: メタ・説明文 */
  --text-base: 1rem;    /* 16px: 本文 */
  --text-lg: 1.125rem;  /* 18px: リード文 */
  --text-xl: 1.25rem;   /* 20px: 小見出し */
  --text-2xl: 2rem;     /* 32px: 大見出し */
  --text-3xl: 3rem;     /* 48px: 大見出し（広幅時） */

  /* フォントサイズ（可変 / fluid） */
  --text-fluid-md: clamp(1.125rem, 1.5vw, 1.375rem);  /* 記事カードタイトル */
  --text-fluid-lg: clamp(1.375rem, 2.2vw, 1.875rem);  /* サイトタイトル */
}
```

- `1rem` = ルート（`<html>`）の文字サイズ。`html` に `font-size` を指定していなければブラウザ既定 16px。
- 可変見出しは `clamp(最小, 画面幅連動, 最大)` で、画面幅に応じて滑らかに変化する。固定スケールで足りる見出しは `--text-*` を、画面幅で伸縮させたい大見出しは `--text-fluid-*` を使う。

### 用途の目安

| 変数 | px 目安 | 主な用途 |
|---|---|---|
| `--text-xs` | 14px | キャプション、フッターのコピーライト |
| `--text-sm` | ~15px | 記事メタ情報、サイト説明文、ページネーション |
| `--text-base` | 16px | 本文（`body`） |
| `--text-lg` | 18px | リード文 |
| `--text-xl` | 20px | 小さめの見出し |
| `--text-2xl` | 32px | 大見出し（404 タイトル等） |
| `--text-3xl` | 48px | 大見出しの広幅時 |
| `--text-fluid-md` | 18→22px | 記事カードのタイトル |
| `--text-fluid-lg` | 22→30px | サイトタイトル |

## 適用ルール

### 1. 文字サイズは必ず変数で参照する

```css
/* OK */
.footer-copyright { font-size: var(--text-xs); }

/* NG: px / 生の値を直書きしない */
.footer-copyright { font-size: 14px; }
```

### 2. px は使わない

新規の `font-size` に px を使わない。既存の px も rem 相当のトークンへ置き換える
（16px 基準では `14px = 0.875rem`、`18px = 1.125rem`、`32px = 2rem`、`48px = 3rem` と等価で見た目は変わらない）。

### 3. 相対指定が適切な箇所は `em` を使う

インラインコードのように「親に対して少し小さく」したい場合は、トークンではなく `em` を使う。

```css
.post-item-content code {
  font-size: 0.9em; /* 親の文字サイズに追従させる */
}
```

### 4. 案件ごとの調整は `:root` だけ編集する

文字サイズのトーンを変える際は各セレクタを触らず、`:root` のトークン値のみ変更する。

## やってはいけないこと

- `font-size` に px を新規で使う
- スケール外の値（`13px`, `17px` など）を直接書く
- 用途と異なるトークンを流用する
